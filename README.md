# Click Speed Tester 🖱️⚡
A browser-based CPS (Clicks Per Second) tester that measures how fast you can click within a chosen time limit.
Features ✨

⏱️ Five timer options: 5, 10, 15, 30 seconds, and 1 minute
🔢 Real-time countdown timer with millisecond precision
🖱️ Live click counter
💧 Ripple animation effect on click area
🏆 Score card at the end with rank, CPS, and total clicks
🎖️ Six rank tiers: Bronze, Silver, Gold, Platinum, Diamond, and Legendary
📱 Fully responsive design for mobile and desktop

**Rank Tiers 🏅**
🥉 BronzeBelow 6
🥈 Silver6 – 8.9
🥇 Gold9 – 11.9
💠 Platinum12 – 14.9
💎 Diamond15 – 19.9
🌟 Legendary20+

**Technologies Used 🛠️**

HTML5
CSS3 (custom properties, CSS Grid, Flexbox, media queries)
Vanilla JavaScript
Font Awesome 7 for icons
